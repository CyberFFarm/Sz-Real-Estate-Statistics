# 深圳房地产预售数据分析系统 · 使用说明

## 文件说明

| 文件 | 作用 | 保留/删除 |
|------|------|-----------|
| `server.py` | 后端代理服务 + SQLite 数据库操作 | ✅ **必须保留** |
| `index.html` | 前端页面（数据总览/项目详情/数据分析） | ✅ **必须保留** |
| `szfdc_data.db` | SQLite 数据库（4387条预售证 + 缓存房源价格） | ✅ **必须保留** |
| `深圳预售许可证.html` | 早期版本，功能已被 index.html 覆盖 | ❌ 可删除 |
| `深圳房地产数据.html` | 早期版本，功能已被 index.html 覆盖 | ❌ 可删除 |
| `amr_scraper.py` | AMR 物价局抓取脚本（暂不可用） | ❌ 可删除 |
| `.DS_Store` | macOS 系统文件 | ❌ 可删除 |

**核心只需要 3 个文件**：`server.py` + `index.html` + `szfdc_data.db`，放到同一个目录下即可运行。

---

## 环境要求

- Python 3.7+
- 无需安装第三方依赖（只用 Python 标准库）

---

## 使用步骤

### 1. 启动服务

```bash
cd 项目目录
python3 server.py
```

启动后显示：
```
深圳房地产数据分析系统 v3.0
==================================================
服务启动: http://localhost:8080
按 Ctrl+C 停止
```

### 2. 访问页面

浏览器打开 `http://localhost:8080`

### 3. 首次使用

- 自动加载数据库中的预售证列表（4387条）
- 点击 **💾 批量缓存房源价格** 按钮，系统自动遍历近5年项目并缓存房源价格到数据库
- 查看进度在状态栏实时显示

---

## 迁移到其他电脑/服务器

### 方式一：直接复制（推荐）

将 3 个核心文件复制到目标电脑同一目录：
```
server.py
index.html
szfdc_data.db
```
然后 `python3 server.py` 启动即可。数据库已包含所有缓存数据。

### 方式二：仅复制代码，重新构建数据库

如果不需要历史缓存数据（或数据库太大），只复制 `server.py` 和 `index.html`：

```bash
python3 server.py
# 首次启动自动拉取预售证列表
# 手动点击"批量缓存房源价格"重建价格数据库
```

### 方式三：局域网共享

```bash
python3 server.py
# 服务监听 0.0.0.0:8080，局域网其他设备可通过 IP 访问
# 例如：http://192.168.1.100:8080
```

---

## 数据更新

- **预售证列表**：点击仪表盘"🔄 刷新列表"重新从住建局拉取
- **房源价格**：点击"💾 批量缓存房源价格"扫描近期项目
- **单个项目**：打开任意项目详情，价格自动存入数据库
- 数据库中的价格不会被后续 API 的 null 值覆盖（价格保护机制）

---

## 端口修改

如需修改端口，编辑 `server.py` 第 27 行：
```python
PORT = 8080  # 改为其他端口
```
