# 深圳房地产预售数据分析平台

基于深圳市房地产信息平台（fdc.zjj.sz.gov.cn）公开数据的预售数据分析系统，支持数据抓取、可视化分析、户型平面图标注及 PPT 简报管理。

## 功能特性

- **📊 数据总览**：区域预售证分布、月度新增趋势图表、区域市场总览（项目数/总房源/有价格/已售/去化率/均价）、开发商与项目发证排名 Top10，支持自定义时间范围筛选
- **📋 预售项目详情**：关键词搜索、区域筛选、CSV 导出、分页浏览
- **🏠 项目详情**：楼栋房源明细、房源状态（可售/已备案/已签认购书等）、备案价格查看、开发商信息
- **🏗️ 平面图工具**：上传户型平面图、标注房源价格标签、支持编辑和保存
- **📑 PPT 简报管理**：为项目上传/预览/下载 PPT 文件（支持多页预览），需管理员密码删除
- **💾 价格保护机制**：房源价格一旦抓取便存入本地数据库，不会被后续 API null 值覆盖
- **📋 系统日志**：记录数据更新历史
- **🔄 批量缓存**：一键缓存近期项目全部房源价格
- **🖼️ 楼盘图片**：支持查看项目效果图等图片资源

## 文件说明

| 文件 | 作用 |
|------|------|
| `server.py` | 后端代理服务 + SQLite 数据库操作 |
| `index.html` | 前端页面（数据总览/项目详情/平面图/PPT管理） |
| `szfdc_data.db` | SQLite 数据库（预售证 + 房源价格 + 平面图数据） |
| `chart.umd.min.js` | Chart.js 图表库（前端图表渲染） |
| `sz_fdc_scraper.py` | 独立数据抓取脚本 |
| `daily_history.json` | 每日数据历史快照 |
| `market_trends.json` | 市场趋势数据 |
| `setup.sh` | 数据库分片组装脚本 |

**核心运行文件**：`server.py` + `index.html` + `szfdc_data.db` + `chart.umd.min.js`

## 环境要求

- Python 3.7+
- 无需安装第三方依赖（仅用 Python 标准库）

## 使用步骤

### 1. 启动服务

```bash
cd 项目目录
python3 server.py
```

启动后显示：
```
深圳房地产数据分析系统 v3.0
==================================================
服务启动: http://localhost:8080
按 Ctrl+C 停止
```

### 2. 访问页面

浏览器打开 `http://localhost:8080`

### 3. 首次使用

- 自动加载数据库中的预售证列表
- 点击 **💾 批量缓存房源价格** 按钮，系统自动遍历近期项目并缓存房源价格到数据库
- 进度在状态栏实时显示

---

## 数据更新

- **预售证列表**：点击仪表盘"🔄 刷新列表"重新从住建局拉取
- **房源价格**：点击"💾 批量缓存房源价格"扫描近期项目
- **单个项目**：打开任意项目详情，价格自动存入数据库
- 数据库中的价格不会被后续 API 的 null 值覆盖（价格保护机制）

## 迁移到其他电脑/服务器

### 方式一：直接复制（推荐）

将核心文件复制到目标电脑同一目录：
```
server.py
index.html
szfdc_data.db
chart.umd.min.js
```
然后 `python3 server.py` 启动即可。

### 方式二：仅复制代码，重新构建数据库

如果不需要历史缓存数据，只复制 `server.py` 和 `index.html`：

```bash
python3 server.py
# 首次启动自动拉取预售证列表
# 手动点击"批量缓存房源价格"重建价格数据库
```

### 数据库分片管理

- `data/` 分片是初始快照，仅用于首次部署
- **更新分片到 Git**：
  ```bash
  rm data/szfdc_data_part_*
  split -b 20m szfdc_data.db data/szfdc_data_part_
  git add data/ && git commit -m "update db snapshot" && git push
  ```

## ⚠️ 数据库说明

- **创建时间**：2026 年 6 月 3 日
- 深圳住建局公开 API 中，房源状态一旦变更为"已备案、已签认购书、已录入合同"，该房源的价格字段即被**服务端抹除**（返回 null/--）
- 因此，**2026 年 6 月 3 日之前已被认购/备案的房源，价格信息在本数据库中缺失**
- 本系统通过价格保护机制（DB 存储 + null 值覆盖保护），确保从今天起新抓取的价格不会被后续状态变更覆盖
- **后续新增的预售项目，只要在房源首次变为"已售"之前打开过一次，价格即可被完整保留**

## 平面图工具

- 打开项目详情 → 切换到"平面图"视图
- 可上传户型平面图（PNG/JPG/WebP）
- 支持在平面图上标注房源编码和价格标签
- 标签可拖拽编辑位置，数据持久化存储

## PPT 简报管理

- 打开项目详情 → 切换到"PPT"视图
- 支持上传/预览/下载 PPT 文件（单文件≤50MB）
- 多页 PPT 可逐页切换预览
- 删除需管理员密码验证（默认密码见 `data/admin_pw.txt`）

## 端口修改

如需修改端口，编辑 `server.py` 第 27 行：
```python
PORT = 8080  # 改为其他端口
```
